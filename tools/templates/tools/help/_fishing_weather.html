使い方

  • GPS追跡 — 「現在地を追跡」ボタンで GPS 位置を地図に表示。移動すると自動でマーカーが更新されます
  • 地図をクリック — 任意の地点をタップ/クリックすると、その場所の天気・潮・波などを取得します
  • 海図レイヤー — 右上のレイヤーボタンで OpenSeaMap の海図（水深線・ブイ・灯台・港）を重ねて表示できます
  • 雨雲レーダー — 雨雲オーバーレイをオンにすると直近の降雨分布を確認できます
  • スポット保存 — 「★保存」で現在の地点をお気に入りとしてブラウザに記録（再訪時に呼び出せます）

表示される情報

  • 天気 / 気温 / 体感 — Open-Meteo による現在の気象
  • 風速・風向 — 矢印で風向、m/s で風速（釣行可否判断の目安）
  • 気圧 — hPa 表示。気圧の低下は食い渋りのサインになることがあります
  • 波高 / 周期 / 向き — 海釣りの可否・安全判断に
  • 海水温 — 魚種の活性判断の目安
  • 潮位グラフ — 24時間の潮位変化。満潮・干潮の時刻
  • 日の出・日の入・マズメ時間 — 朝マズメ / 夕マズメの目安時間帯
  • 月齢 — 大潮・中潮などの潮まわり参考

釣りのコツ（参考）

  • マズメ時 — 日の出前後・日の入前後は魚の活性が上がりやすい時間帯
  • 潮の動く時間 — 満潮/干潮の前後は潮流が動き、魚が捕食行動に入りやすい
  • 波高の目安 — 波高1.5m以上は堤防・磯での釣行は注意。安全第一で
  • 気圧の変化 — 急激な低下時は食いが落ちることが多い

海図記号の読み方(🌊海図レイヤー ON 時)

  • ⭕ 小さな円(単独) — 岩礁・暗礁・隠れ根。船には危険ですが、根魚(メバル・カサゴ・ソイ)やロックフィッシュの好ポイント
  • ⭕ 大きめの破線円 — 浅瀬・砂州・錨地(アンカレッジ)の範囲。遠浅ならキス・カレイ、駆け上がりは青物回遊に期待
  • 🔺 菱形・三角マーク — ブイ・浮標(航路標識)。周辺は潮目ができやすく回遊魚ポイント(ただし航路ブイへの係留は違法)
  • ⭐ 星マーク・光付き記号 — 灯台・灯標。岬や港の入口に設置
  • ⚓ 錨マーク — 錨泊地。船舶の停泊海域

線の意味

  • 細い点線(等深線) — 同じ水深を結んだ線。釣りで最重要。線が密集していれば急深(駆け上がり=ブレイク)、緩いなら遠浅
  • 等深線の数字 — 水深(メートル)。10m・20m・50m・100m などで描かれる
  • 長い点線(海底ケーブル / パイプライン) — 投錨・底引き禁止区域。オモリが引っ掛かると法令違反+仕掛けロスト。底物釣りは避ける
  • 破線で囲まれた区域 — 漁業制限区域・禁漁区・錨泊禁止区域など。場所により規制内容が異なる
  • 二重線・太い線 — 推薦航路・分離通航帯。大型船が通行する海域
  • 実線の境界線 — 港湾区域・行政界(県境・領海線など)

等深線から読む釣り場の特徴

  • 線が密集した岬の先端 — 急深で潮が早い。青物(ブリ・ヒラマサ)・回遊魚のショアジギポイント
  • 等深線が湾状に入り込む場所 — 海底に谷・ミオ筋があるサイン。魚の回遊通路になりやすい
  • 岩礁マーク + 等深線の変化 — 根魚の住処。ロックフィッシュの一級ポイント
  • 緩やかな等深線が続く砂浜 — 遠浅サーフ。投げ釣り・ヒラメ・マゴチ狙い

注意事項

  • 位置情報の許可が必要 — GPS追跡にはブラウザの位置情報許可が必要です（HTTPS 接続必須）
  • データは予報値 — 海洋情報は予測値です。実際の現地状況と異なることがあります
  • 安全第一 — 海況が悪い場合は無理をせず釣行を中止してください
