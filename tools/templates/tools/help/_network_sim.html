基本操作

  • 機器配置 — パレットから機器（PC・ルーター・スイッチ等）をクリックし、キャンバス上でクリックして配置
  • 移動 — 配置済みの機器をドラッグで移動
  • ケーブル接続 — 「ケーブル接続」ボタンをクリック後、2つの機器を順にクリックして接続
  • 削除 — 「削除」ボタンをクリック後、機器またはケーブルをクリックして削除
  • 全クリア — キャンバス上の全機器・接続を一括削除

プリセット構成

  • シンプル — PC-Router-PC の基本構成
  • DNS + Web構成 — DNSサーバーとWebサーバーを含む構成
  • 冗長ルーター — 2経路の冗長ルーター構成
  • 企業ネットワーク — ファイアウォール + DMZ を含む企業構成
  • データセンター — コア・アグリ・アクセスの3層構成
  • メールシステム — SMTP中継によるメール送受信構成
  • 企業フル構成 — DHCP + DNS + Mail + LB を含む総合構成

ターミナル

  • 起動方法 — PCを右クリック →「ターミナルを開く」
  • help — 使用可能なコマンド一覧を表示
  • ping <IP|ホスト名> — ICMP Echo を送信して疎通確認（パケットの流れをアニメーション表示）
  • traceroute <IP|ホスト名> — 宛先までの経路をホップごとに表示（tracert でも可）
  • nslookup <ドメイン> — DNS名前解決（dig でも可）
  • curl <URL|IP> — HTTP リクエストを送信（wget でも可）
  • ifconfig — インターフェース情報を表示（ip でも可）
  • arp — ARPテーブルを表示
  • route — ルーティングテーブルを表示
  • netstat — 接続状態を表示
  • dhclient — DHCPでIPアドレスを自動取得
  • mail <user@domain> — メールを送信
  • ftp <IP|ホスト名> — FTP接続
  • ntpdate <IP|ホスト名> — NTPサーバーと時刻同期
  • tcpdump [キャプチャ名] — パケットキャプチャログを表示
  • clear — ターミナル画面をクリア

障害シミュレーション

  • 故障設定 — 機器やケーブルを右クリック →「故障を設定」
  • 故障中の機器は赤く表示され、パケットが転送されなくなります
  • 冗長構成で故障時のフェイルオーバーを確認できます

プロパティパネル

  • 機器をクリックすると、IPアドレスやルーティングテーブルなどの設定を確認・編集できます
