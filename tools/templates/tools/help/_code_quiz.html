使い方

  • 言語を選択して「出題」ボタンを押すと問題が表示されます
  • 4つの選択肢から正解を1つ選んでください
  • 回答後に解説が表示されます

問題の形式

  • 穴埋め — コードの空欄に入る正しい記述を選ぶ
  • 選択 — 正しい説明や記述を選ぶ
  • 出力予測 — コードの実行結果を予測する
  • エラー発見 — コード内の誤りを見つける

対応言語

  • HTML — タグ・属性・構造に関する問題
  • CSS — セレクタ・プロパティ・レイアウトの問題
  • JavaScript — 構文・DOM操作・非同期処理の問題
  • Python — 構文・データ型・組み込み関数の問題

正解率

  • ページを離れるまでのセッション中の正解率が表示されます
  • ページを離れるとリセットされます
